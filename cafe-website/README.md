# Lumina Café - Local Business Website

A modern, responsive, and SEO-optimized website built for a local café business to establish a strong digital presence.

## 🚀 Features

- **Home Page**: Attractive hero section with a clear Call to Action (CTA).
- **About Section**: Tells the business's story, building trust and connection.
- **Services Section**: Showcases what the café offers (coffee, pastries, tea).
- **Gallery**: Beautiful grid to highlight the café's atmosphere.
- **Contact Page**: Includes a contact form, direct email/phone details, and a Google Maps embed.
- **Floating WhatsApp Button**: Allows instant communication, highly converting for local businesses.
- **Responsive Design**: Flawless experience on mobile, tablet, and desktop.
- **SEO Optimized**: Standard meta tags and semantic HTML for search engines.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom variables, Flexbox, Grid, and smooth animations (no external frameworks used).
- **JavaScript**: Interactive mobile navigation, sticky header, and form handling.

## 🌐 How to Host the Website

To get this website live and accessible to your customers, follow these simple steps using a free hosting provider like **Netlify**, **Vercel** or **GitHub Pages**.

### Option 1: Using Netlify (Easiest)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Create an account or log in.
3. Drag and drop the `cafe-website` folder into the designated area.
4. Netlify will instantly generate a live link (e.g., `https://your-site-name.netlify.app`).
5. (Optional) You can assign a custom domain like `www.luminacafe.com` in the Netlify settings.

### Option 2: Using GitHub Pages
1. Push this project to a new repository on GitHub.
2. Go to your repository **Settings**.
3. Scroll down/click on **Pages** in the left sidebar.
4. Under "Source", select the `main` branch and click **Save**.
5. Your site will be published at `https://[your-username].github.io/[repo-name]`.

## 📈 How This Website Helps the Business Grow

1. **Increased Visibility (SEO):** The website ensures that when locals search for "coffee near me," "cafe," or "pastries" on Google, the business can show up with proper SEO and the embedded local map.
2. **Instant Conversions via WhatsApp:** The floating WhatsApp button removes friction. A customer can book a table, ask for opening hours, or arrange a pickup order with a single tap.
3. **Credibility & Trust:** A premium, modern design signals quality. Customers relate the quality of the website to the quality of the coffee and service provided.
4. **Information Hub:** Customers have immediate access to opening hours, location, and the menu, saving staff time answering repetitive phone calls.
5. **Brand Storytelling:** The 'About' gallery and section build a personal connection with customers, turning first-time visitors into regulars.
